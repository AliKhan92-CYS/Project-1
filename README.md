 Pac-Man Game (Java OOP Project)
 **Description**
This is a simple Pac-Man style game I made in Java for my semester project. The game is built using basic OOP concepts and Java Swing for graphics. You control Pac-Man using arrow keys, collect food, and try to avoid the ghosts. As you keep playing, more enemies are added which makes the game a bit challenging.
I mainly made this project to practice how classes work together, how game loops run, and how to handle movement and collisions in Java.
 **Features**
•	🟡 Custom Pac-Man character with animated mouth and directional movement
•	👁️ Dynamic eye positioning based on movement direction
•	🍽️ Food (dots) system with score tracking
•	👻 Multiple enemies (ghosts) with smooth random movement
•	😈 Custom ghost design with animated appearance
•	📈 Difficulty increases over time (new ghosts spawn gradually)
•	🧱 Maze-based movement with proper wall collision detection
•	🎮 Keyboard controls with smooth movement (hold vs press support)
•	🏁 Win condition when all food is collected
•	💀 Game over on collision with enemy
** Technologies Used**
•	Java
•	Java Swing (GUI & Graphics)
•	OOP Concepts (Classes, Objects, Encapsulation, Relationships)
**📁 Project Structure**
Game.java        → Entry point  
GameFrame.java   → Game window (JFrame)  
GamePanel.java   → Game loop, rendering, controls  
Pacman.java      → Player logic and movement  
Ghost.java       → Enemy behavior and movement  
Maze.java        → Map, walls, food system  
**Controls**
•	⬆️ Up Arrow → Move Up
•	⬇️ Down Arrow → Move Down
•	⬅️ Left Arrow → Move Left
•	➡️ Right Arrow → Move Right
👉 Hold key for continuous movement
👉 Tap key for small movement
**How to Run**
1.	Clone the repository
2.	Open the project in any Java IDE (IntelliJ / Eclipse / VS Code)
3.	Compile all .java files
4.	Run Game.java
