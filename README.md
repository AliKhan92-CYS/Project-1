# 🟡 Pac-Man Game (Java OOP Project)

## 📌 Description
This is a simple **Pac-Man-style game** built using **Java** and **Object-Oriented Programming (OOP)** concepts. The game uses **Java Swing** for graphics and rendering.

In this game, the player controls Pac-Man using arrow keys, collects food, and avoids ghosts. As the game progresses, more enemies spawn, increasing the difficulty level.

This project was created to practice core programming concepts such as OOP design, game loops, collision detection, and event-driven programming in Java.

---

## 🎮 Features
- 🟡 Custom Pac-Man character with animated mouth and directional movement  
- 👁️ Dynamic eye movement based on direction  
- 🍽️ Food (dots) collection system  
- 📊 Score tracking system  
- 👻 Multiple ghosts with smooth movement  
- 😈 Custom ghost design with animation  
- 📈 Increasing difficulty over time (new ghosts spawn gradually)  
- 🧱 Maze-based collision detection system  
- 🎮 Smooth keyboard controls (hold & tap support)  
- 🏁 Win condition when all food is collected  
- 💀 Game over on collision with ghosts

## 📁 Project Structure
Game.java        → Entry point  
GameFrame.java   → Game window (JFrame setup)  
GamePanel.java   → Game loop, rendering, controls  
Pacman.java      → Player logic and movement  
Ghost.java       → Enemy behavior and AI movement  
Maze.java        → Map design, walls, and food system  

---

## 🎮 Controls
- ⬆️ Up Arrow → Move Up  
- ⬇️ Down Arrow → Move Down  
- ⬅️ Left Arrow → Move Left  
- ➡️ Right Arrow → Move Right  

👉 Hold key for continuous movement  
👉 Tap key for small movement  

---

## 🚀 How to Run
1. Clone the repository  
2. Open the project in IntelliJ / Eclipse / VS Code  
3. Compile all `.java` files  
4. Run `Game.java`
