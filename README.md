# 🟡 Pac-Man Game (Java OOP Project)

## 📌 Description
This is a simple **Pac-Man-style game** built using **Java** and **Object-Oriented Programming (OOP)** concepts. The game uses **Java Swing** for graphics and rendering.

In this game, the player controls Pac-Man using arrow keys, collects food, and avoids ghosts. As the game progresses, more enemies spawn, increasing the difficulty level.

This project was created to practice:
- Object-Oriented Design (OOP)
- Game loop implementation
- Collision detection
- Event-driven programming in Java

---

## 🎮 Features
- 🟡 Custom Pac-Man character with animated mouth and directional movement  
- 👁️ Dynamic eye movement based on direction  
- 🍽️ Food (dots) collection system  
- 📊 Score tracking system  
- 👻 Multiple ghosts with smooth movement  
- 😈 Custom ghost design with animation  
- 📈 Increasing difficulty (new ghosts spawn over time)  
- 🧱 Maze-based collision system  
- 🎮 Smooth keyboard controls (hold & tap support)  
- 🏁 Win condition when all food is collected  
- 💀 Game over on collision with ghosts  

---

## 🛠️ Technologies Used
- Java  
- Java Swing (GUI & Graphics)  
- Object-Oriented Programming (OOP)
